# ReactNative

原生应用开发

# 前言

我是先学的Vue。react并没有学过。

为什么不用weex，因为文档太少。reactnative文档多，社区比较活跃。

所以这里只会记录，怎么做，并没有为什么。

平台linux，Android。

# 正文

从安装开始。甩手就是一片官方文档。已经假设已经安装好了Android sdk,如果没有，直接安装android studio。一切解决。

react-native init projectname
cd projectname
reactnative run-android
  • 如果没有sdk安装路径，请在~/bashrc指定，一般实在~/Android/Sdk。这部分文档中有。
  • linux可以命令行安装platform-tools。而android studio下载时候会帮你下载好。这两部分可能会冲突，版本号不一致。做法就是把~/Android/Sdk某个文件夹下复制到~/usr/bin下即可

大致是没有问题了。

# Errors

  • gradle版本太低，因为react-native要求高

  • com.android.support:support-v4 error
    dependencies {
compile "com.android.support:appcompat-v7:27.1.0" // 后面版本高一点就行
  • resource没有找到 - app/build.gradle里面compileSdkVersion 27 buildToolsVersion "26.1.0"因为要和电脑上的版本一直。
  • invariant violation element type is invalid - 这是因为babel问题，app.js需要export defalut class。这里要注意：
    class  BadInstagramCloneApp extends Component 
//..
export BadInstagramCloneApp 记得分开写
  • require txt等文件，在根目录下追加文件rn-cli.config.js
    module.exports = {
    getAssetExts() {
        return ['pb', 'txt']//pb txt都是文件后缀名
    }
}
